2. 	Compilación y Ejecucion
	
	* Para compilar escriba:
		make comp

	* Esto le generará un archivo out

	* Para ejecutar debe escribir
		make run file=<direccion donde esta su instancia>

3.	Ejemplo

	make comp

	* Si es que mi instancia esta en una carpeta llamada instancias y se llama instancia1.txt, entonces debo realizar lo siguiente

		make run file=instancias/instancia1.txt

	* Esto además creare un archivs instancia.txt.ans, en la misma carpeta instancias con la respuesta del programa
