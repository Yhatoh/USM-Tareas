Como correr el programa:

1.- Tener python3 instalado en el computador 
    que se desea correr.
2.- Tener el archivo checker.py en la misma direccion
    que el archivo (o los archivos) que posean el
    codigo de "LOLCODE", de preferencia en formato
    "txt"
3.- Correr el programa
4.- Se le solicitara que introduzca el nombre del 
    archivo que se quiera analizar
5.- Introduzca el nombre del archivo, asegurarse de
    tambien añadir la extension del archivo, ejemplo
    "prueba.txt"
