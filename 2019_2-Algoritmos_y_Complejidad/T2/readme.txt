Para la compilacion automatica puedo hacer los siguientes comandos:
	Para el problema 1:
		make prob1-bf (que compila el problema que utiliza fuerza bruta)

		make prob1-dc (que compila el problema que utiliza dividir)

	Para el problema 2:
		make prob2

Cada compilacion genera un archivo "out", por lo que obviamente corre un programa y de ahi compila el otro.

Para la ejecucion hazlo de la siguiente forma:
	./out < input.txt > output.txt

	asi funcionara de pana banana
